Insert PlayerID for each goal they score.
For example: 2 Continue Game, 3 Continue Game, 2 Continue Game, 6 Continue Game, 2 Save Game
This would say Player 2 scored 3 goals, player 3 scored 1 goal, and player 6 scored 1 goal.
This would then be saved for the game information, and the winner and loser will be determined with the amount of goals scored for each team. If there is a tie, no team will gain a win or loss. The game can be edited later to make sure a team wins, and loses.
When PlayerID then Coninue Game is selected, the goal is still saved completely.
The only real difference between Continue Game and Save Game is that Continue Game is selected with the intent to add more goals, while Save will put the team back to the Teams Menu.
However, a team may click Continue Game then Back, and the game will be saved still. Again, Save is more convient when the game is completed, so it will take the team back to the teams menu. By using continue, goals will still be saved.
